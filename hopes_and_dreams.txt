The Grand Vision:

A web app that allows a user to configure their list from a selection of metadata filters, determine randomization style, and then start a game session that generates a new effect on demand and keeps a log of history for that game. Nothing is persisted outside the browser session.

JSON file contains effect list:
Name, ShortDescription, FullDescription, CanonicalNumber, DisplayNumber, Players [Duel, Multi], Inclusion [Vengeance, Micro, Lite, Exemplar, Legacy, Minigame, Physical], Exemplar [Main, Tokenizer, Complexity, Sound-Alike, Prodigal, Beat, University, Toxin, Bad Things], School [Abjuration, Chronomancy, Conjuration, Divination, Enchantment, Evocation, Illusion, Necromancy, Transmutation], Duration [Instant, Turn, Round, Ongoing], Rarity [Common, Uncommon, Rare, Mythic], Component [TokenX, CounterX]
